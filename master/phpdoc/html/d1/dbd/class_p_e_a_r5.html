static

If you have a class that's mostly/entirely static, and you need static properties, you can use this method to simulate them.

Eg. in your method(s) do this: $myVar = &PEAR5::getStaticProperty('myclass', 'myVar'); You MUST use a reference, or they will not persist!

Parameters
string$classThe calling classname, to prevent clashes
string$varThe variable to retrieve.
Return values
mixedA reference to the variable. If not set it will be auto initialised to NULL.
